# KSOFT-BY-KOICH


## 🛠️ Зависимости и скачивание ядра Xray (Обязательно)

Графический интерфейс **KSOFT** управляет мощным низкоуровневым туннелем через бинарное ядро **Xray-core**. Без него подключение к серверам работать не будет.

### 1. Скачивание Xray-core
Перейдите в официальный репозиторий разработчиков на GitHub в раздел релизов:
👉 **[Скачать Xray-core с GitHub Releases](https://github.com/xtls/xray-core/releases)**

В списке файлов (блок **Assets** у самого свежего релиза) выберите архив для вашей версии Windows:
* **`Xray-windows-64.zip`** — Подходит для большинства современных 64-битных систем (Windows 10 / 11).
* **`Xray-windows-32.zip`** — Для старых 32-битных версий Windows.
* **`Xray-windows-arm64.zip`** — Для устройств на мобильных процессорах ARM (например, Microsoft Surface на базе Snapdragon).

### 2. Установка
1. Распакуйте скачанный `.zip` архив.
2. Скопируйте из него файл **`xray.exe`** (остальные файлы из архива можно не трогать).
3. Перенесите `xray.exe` в корневую папку, где находится ваш скрипт `main.py` (или скомпилированный `KSOFT.exe`).
